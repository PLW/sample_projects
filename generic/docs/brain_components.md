
### Brain component outline

Below is a concise, species‑agnostic map of the common large‑mammal
brain plan: humans, chimps, bonobos, orcas, dolphins, and elephants.
What differs across species is relative size, connectivity density,
and sensory emphasis.

--- 

## Major components and what they do

* Cerebral cortex (neocortex + insula)
  * Primary areas:
    * visual
    * auditory
    * somatosensory
    * motor—encode basic
    * features and drive movements
  
  * Association networks:
    * fronto‑parietal 
      - goal setting
      - working memory
    * temporo‑parietal/temporal 
      - social cognition
      - semantic knowledge
    * insula 
      - interoception
      - taste
      - pain integration

* Hippocampal formation (hippocampus, dentate gyrus,
    entorhinal cortex)
  * episodic/spatial memory
  * sequence learning
  * consolidation during sleep
  * map environments and events

* Amygdala and extended limbic circuit (amygdala, cingulate,
    orbitofrontal, ventral striatum)
  * valence/salience tagging
  * threat and reward learning
  * social/emotional decision signals

* Basal ganglia (striatum—caudate/putamen/nucleus accumbens,
    pallidum, subthalamic nucleus, substantia nigra)
  * action selection and habit formation
  * reinforcement learning]
  * based on dopamine.

* Thalamus (specific and association nuclei)
  * Cortical hub/relay gates sensory flow:
    * LGN for vision
    * MGN for audition
  * synchronizes cortical rhythms

* Cerebellum (hemispheres + vermis, deep nuclei)
  * predictive control and timing for movement
  * also contributes to language
  * planning
  * cognition via the same loop motif.

* Hypothalamus
  * homeostasis (hunger, thirst, temperature)
  * endocrine control (pituitary)
  * sleep–wake regulation
  * survival behaviors.

* Brainstem and tectum (midbrain, pons,
    medulla superior/inferior colliculi)
  * arousal
  * autonomic control
  * orienting reflexes
  * cranial‑nerve
  * sensorimotor interfaces

* Neuromodulatory systems (broadcast networks)
  * dopamine (VTA/SNc: learning, vigor),
  * norepinephrine (locus coeruleus: alerting, exploration),
  * serotonin (raphe: mood, patience),
  * acetylcholine (basal forebrain/PPN: attention, plasticity),
  * histamine and orexin (arousal, wakefulness).

* Olfactory system (olfactory bulb → piriform/limbic cortex)
  * Smell; its size varies widely (very large in elephants; strongly
    reduced/absent in many toothed whales).

* Major long‑range tracts
  * Corpus callosum and anterior commissure
        → interhemispheric exchange
  *  cingulum
        → limbic
  *  uncinate
        → orbitofrontal–temporal
  *  superior/inferior longitudinal fasciculi and arcuate
        → fronto‑parietal/temporal integration
  *  fornix
        → hippocampus ↔ hypothalamus/mammillary bodies

## Canonical circuits (how they’re wired)

* Thalamo‑cortical loops (TC)
  * Flow:
     Sensory input → thalamus
                   → primary cortex
                   → higher association cortex
                   → feedback to thalamus/earlier cortices.  
  * Role: Hierarchical feature extraction with top‑down predictions
    (attention, expectation).

* Cortico‑basal_anglia‑thalamo‑cortical loops (CBGTC)
  * Parallel channels (motor, oculomotor, associative/cognitive, limbic).
  * Role: Select "what to do next", gates working memory, shapes habits
    via dopamine‑based credit assignment.

* Cortico‑ponto‑cerebello‑thalamo‑cortical loops (CPCTC)
  * Cortex → pontine nuclei → cerebellum → thalamus → cortex.  
  * Role: Build forward models to reduce error in movement and thought;
    fine‑tunes timing and sequencing.

* Hippocampal–limbic memory circuit (HLM Papez/extended)
  * Hippocampus → fornix
                → mammillary bodies/anterior thalamus
                → cingulate/parahippocampal
                → hippocampus.  
  * Role: Bind "what‑where‑when" experiences and integrate
    emotion into memory.

* Brainstem neuromodulatory "broadcast”
  * Global gain control that reweights all the above loops depending on
    context (threat, novelty, opportunities, fatigue).

## Typical information flow (very high‑level)

1. Sense and gate:
   Peripheral receptors → brainstem/colliculi
                        → thalamus (except much olfaction).  

2. Decode and integrate:
   Thalamus → primary cortex
            → association networks;
   continuous top‑down predictions modulate incoming signals.  

3. Select and refine actions:
   Association/motor cortex ↔ basal ganglia (select)
   and ↔ cerebellum (predict/refine) → thalamus → back to cortex.

4. Remember and evaluate: Cortical states ↔ hippocampus
   (episodic/spatial) and ↔ amygdala/orbitofrontal (value/social
   meaning).  

5. Act and regulate: Hypothalamus/brainstem set
   autonomic/endocrine context; spinal cord/cranial nuclei execute
   movements and vocalizations.  

6. Modulate: Dopamine/NE/5‑HT/ACh
   tune plasticity, exploration/exploitation, attention, and mood across
   all loops.

## Species‑typical emphasis (same plan, different weights)

* Humans and great apes (chimp/bonobo):
  * Large association cortex (fronto‑parietal and temporal)
  * Extensive white‑matter supporting
    * multi‑step planning,
    * tool use, and
    * complex vocal/social communication.

* Cetaceans (orca, dolphin):
  * dominant auditory pathways,
  * expanded paralimbic/insular–cingulate regions,
  * supporting vocal learning, echolocation, and social processing,
  * olfaction reduced in toothed whales.  

* Elephants:
  * hypertrophied olfactory system,
  * rich somatosensory maps (trunk),
  * strong temporal‑limbic networks for spatial/social memory,
  * large cerebellum supporting precise, multi‑effector control.

--- 

## Summary

These large mammals share the same core modules and the same
loop architecture:
  * thalamo‑cortical hierarchies for representation,
  * basal ganglia loops for selection,
  * cerebellar loops for prediction,
  * hippocampal–limbic loops for memory and value, and
  * neuromodulatory broadcasts that retune the whole system
    to the current demands.

The differences you notice between species largely reflect which
inputs are most privileged (vision, audition, smell, touch) and
how much "bandwidth” each loop gets.
